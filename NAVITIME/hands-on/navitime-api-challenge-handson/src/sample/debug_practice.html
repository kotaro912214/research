デバッグ練習

デバッグ練習の手順は下記の通りです。
わからない、話においてかれた場合は近くの社員に聞いてください。

  1. 画面上のどこかで右クリック
  2. 検証を選択

  3. 出てきた検証画面内のSourcesをクリック
  4. 現在開いているファイルを指定
  5. 63行目 console.log("クリック回数：" + count); にブレークポイントをつくる
    (行番号をクリックして数字が青く選択されればOK)

  6. 下のボタンをクリックしてみましょう

  7. Consoleに count と入力し、Enterキーを押しましょう
  8. 整数が表示されることを確認

  9. 処理をひとつ進めましょう
  10. Consoleに クリック回数: X と表示されていることを確認しましょう

  11. 処理を最後まで進めましょう
  12. ポップアップが出てくることを確認

早く終わった人はやってみよう

レイアウト崩れの原因調査、cssを設定したらどのように効くかを確認してみよう

  1. 画面上のどこかで右クリック
  2. 検証を選択

  3. 左上の矢印をクリック
  4. その状態で各htmlタグ要素の上にカーソルを合わせてみよう
  5. その状態で下の箱をクリック
  6. .debug-div の{}の中にcssの記載をしてみよう

    7. border-color: blue; を追記
  7. 枠線の色が変わったらok
※ カーソルを動かすごとに色々出てくるのが嫌な場合は始めに押した矢印をクリックすれば元に戻ります。
